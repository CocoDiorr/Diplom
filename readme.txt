На этой ветке добавлены программы обработки пакетов для режима ПКС коммутации.
1) sdn-l2-switch.asm
 - Таблица потоков: дерево промежуточного формата программы.
 - Файлы деревьев: tree_1.asm и tree_2.asm в директории example/Tree.
 - Тестовые .pcap файлы в директории example/sdn-l2-switch.
2) sdn-l2-switch-vlan.asm
 - Таблица потоков: дерево из деревьев промежуточного формата программы.
 - Файлы деревьев в директории example/Tree.
 - Тестовые .pcap файлы в директории example/sdn-l2-switch-vlan вместе с readme.txt,
   в котором указаны правильные результаты обработки каждого пакета.
3) sdn-l2-switch-vlan-union.asm
 - Таблицы потоков объединены в одну общую таблицу, которая реализована в виде дерева
   из деревьев промежуточного формата программы.
 - Файлы деревьев в директории example/Tree_union.
 - Тестовые .pcap файлы в директории example/sdn-l2-switch-vlan вместе с readme.txt,
   в котором указаны правильные результаты обработки каждого пакета.
4) sdn-l2-switch-vlan-labeled-trees.asm
 - Таблицы потоков: дерево представлено в виде набора таблиц.
 - Файлы деревьев в директории example/Tree.
 - Тестовые .pcap файлы в директории example/sdn-l2-switch-vlan вместе
   с readme.txt, в котором указаны правильные результаты обработки каждого пакета.

Для поддержки макросов, состоящих из более чем одной команды, в парсер добавлен разделитель ';'.

Дополнительно реализованы
- команды:
  1) setsz
  2) setmask
  3) setxmask
  4) loadmeta
  5) storemeta
  6) loadoff
  Примечание: Данные команды ЗАТИРАЮТ значение в регистре аккумуляторе!
- директивы:
  1) tree_in
  2) tree_lpm

Счетчики пока не поддерживаются, но в программах работа с ними обозначена макросом COUNTER_INC(x).
